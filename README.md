# lingxi-head-assets

> 把head标签里面的资源引用内容提取出来，然后以内联方式嵌入


### Usage Examples

```js
grunt.initConfig({
  lingxi_head_assets: {
    files: {
      src: [
        './test/target/*.html'
      ]
    },
  },
});
```
